# Анастасия | Системный аналитик

Проектирование архитектуры информационных систем, разработка технической документации и проектирование интеграционных решений. 
Специализируюсь на анализе сетевой безопасности и High-Level Design.

### 🛠 Технологический стек:
*   **Моделирование:** UML (Sequence, State, Use Case), DFD, ERD.
*   **Интеграции:** Проектирование API (REST, SOAP), описание взаимодействия микросервисов.
*   **Данные и разработка:** SQL, Python (FastAPI, Pandas), Git.
*   **Сети:** Стек TCP/IP, модель OSI, безопасность сетевого периметра.
*   **Документация:** Разработка SRS, ТЗ (ГОСТ 34/19), матрицы трассировки требований.

### 🏛 Инженерная база:
*   **Школа 21 (эко система Сбер):** Системный и бизнес анализ. Интенсивная подготовка по методологии peer-to-peer, проектирование архитектуры и работа с ТЗ.
*   **Russia Cup (Changellenge):** Аналитика и оптимизация процессов в рамках федеральных кейс-чемпионатов.
